# KubeJS Goety
闇の魔術が新たな世界の扉を開きます…  

## はじめに
KubeJS Goetyはその名の通り、KubeJSとGoety間の連携を高めるアドオンです。  
主に以下のことができます。  

- 独自の祭壇の作成  
- 醸造の作成  
- 祭壇や醸造レシピ、粉砕フォーカスのレシピ等のカスタマイズ  

詳細はそれぞれのセクションで解説しています。  

## Goety Revelationについて
Goety RevelationはGoetyを包括的に強化するエンドコンテンツModです。  
KubeJSと独自に連携をしているわけではございませんが、**データパックでKubeJS Goety**のようなことを行えます。お好きなほうをお選びください。
