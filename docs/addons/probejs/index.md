# ProbeJS
VSCodeでKubeJSスクリプトを書くときに必須級と言われるアドオンです。  
**ここに記載されている情報は1.20.1版のものです。1.18.2以前や1.21以降とは異なる場合がございます。**

## 導入

ProbeJSは主にVSCode上でKubeJSのコーディングを補助するアドオンです。全人類入れよう。  
1. KubeJSが導入されたインスタンスにProbeJSを導入する。  
2. VSCodeの拡張機能である「[ProbeJS](https://marketplace.visualstudio.com/items?itemName=Prunoideae.probejs)」を導入する。VSCode拡張機能の導入方法はggr。  
3. マインクラフトのワールドに参加し、`/probejs dump`を実行する。初回参加時は自動で行ってくれます。  
4. VSCodeを**インスタンスフォルダ(minecraftフォルダ)**で開く。主にconfigフォルダやmodsフォルダの親フォルダです。  
5. "`jsconfig.json`を読み込み中"が終了次第、ProbeJSによる補完が効くようになります！  

## 使用方法

著者も使いこなせてないペーペーです。そのため、不確かな情報や効率的でない情報が載っている場合がございます。ご了承ください。  

スクリプトを書き始めようとしたとき、補完が効いているのがわかるでしょう。この機能は偉大で素晴らしく、KubeJSを書く時には手放せないものです。まぁ、ちょっとしたアイテムやレシピの追加であればいらないかもしれませんが、高度なスクリプトを書く際は必須と言えます。
